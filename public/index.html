A Telegram bot that helps you store and organize your receipts in Notion.

About This Bot

This bot allows you to:

  • Upload receipt images or PDFs via Telegram
  • Automatically extract receipt details
  • Store all data in your Notion database
  • Track expenses with dates and amounts

How to Get Started

1 Add the bot to Telegram

Click the button below to start chatting with the bot:

Open in Telegram
2 Set up your Notion integration

Create a new integration in Notion and get your API key.

Create a database in Notion with the following properties:

  • Name (title)
  • Date (date)
  • Type (select)
  • Status (select)
  • Amount (number)
  • Receipt URL (url)
3 Start sending receipts

Simply send a photo of your receipt or a PDF to the bot, and it will be automatically processed and added to your Notion database.
